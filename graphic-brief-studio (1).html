🔑 שלב 0 — הגדרת API Key

1. היכנס ל-platform.openai.com
2. לחץ על "API Keys" ← "Create new secret key"
3. הדבק את המפתח בתיבה בסרגל השמאלי ולחץ "שמור"
המפתח נשמר רק בדפדפן שלך ולא נשלח לשום שרת חיצוני.

✏️ שלב 1 — אפיון הבריף

מלא את כל השדות בדף האפיון. ככל שתהיה ספציפי יותר — כך התוצאה תהיה טובה יותר:
• שם מותג + תעשייה — נותן context לAI
• פורמט — משפיע ישירות על הגדלים ב-DALL·E
• נושא ומסר — מה נראה בגרפיקה
• סגנון + צבעים — מכוון את האסתטיקה

⚡ שלב 2 — עיבוד ויצירה

Claude מעבד את הבריף → בונה פרומפט מקצועי בשפת DALL·E → שולח ל-API.
תוצאה בדרך כלל תוך 15-30 שניות. אפשר ללחוץ "בנה פרומפט מחדש" לגרסה שונה.

💡 טיפים להצלחה

• טקסט בגרפיקה: DALL·E לא מוצלח בטקסט — הוסף אותו ב-Canva/Figma אחר כך
• לוגו: ציין בהוראות "ללא לוגו" ותוסיף אותו ידנית
• גרסאות: לחץ "נסה שוב" 2-3 פעמים לגרסאות שונות ובחר הטובה
• שמור לגלריה: שמור את כל הגרסאות הטובות לפני שסוגר
